# Do Zero ao Herói do Código | Liderança em Tecnologia

Repositório destinado ao armazenamento de exercícios, desafios e projetos desenvolvidos durante o curso livre de Liderança em Tecnologia, ofertado pela AFIE TECH PARK em Ponta Grossa.
## Sobre o Curso

O programa tem carga horária de 320 horas e duração de 5 meses (Fevereiro a Julho de 2026), focando em preparar talentos para o mercado de desenvolvimento web e mobile.

    Instituição: AFIE TECH PARK (Associação de Fomento à Inovação e Educação Empreendedora).

    Metodologia: PBL (Project Based Learning / Aprendizagem Baseada em Problemas).

    Status atual: 🟢 Nível Básico (Módulo I).

## Tecnologias e Conteúdos

De acordo com o cronograma do curso, este repositório abrangerá:

    Fundamentos de Programação: JavaScript.

    Programação Intermediária: Node.js, Express e MongoDB.

    Desenvolvimento NoCode: FlutterFlow.

## Estrutura do Repositório

Atualmente, o repositório está organizado de forma linear para o nível básico. Futuramente, será segmentado por complexidade.

## Requisitos de Participação

Para o desenvolvimento dessas atividades, sigo os critérios estabelecidos no edital:

    Raciocínio lógico e noções de inglês.

    Perfil colaborativo e resiliente.

    Disponibilidade para estudos presenciais.
